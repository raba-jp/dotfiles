# dotfiles

## Installation

### NixOS

```bash
sudo nixos-install --flake github:raba-jp/dotfiles#define7
```

Or

```bash
sudo nixos-install --flake github:raba-jp/dotfiles#xps13
```

## Manual Installation

**Only macOS**

Import Better Touch Tool preset.

File path is `dotfiles/files/Default.bttpreset`.
